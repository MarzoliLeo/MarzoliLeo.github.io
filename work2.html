002 : Glitch Garden

This game idea was inspired by the famous game "Plant vs Zombie". In Glitch Garden there are levels and each one make you unlock a new plant you can use to defend from enemies. You can put an enemy only if you have enough points, which are generated by the sunflowers you put in the field. If you survive the waves of mobs you win the level and go next.

This game is offline, for computer and WebGL. Developed in 3 weeks.
